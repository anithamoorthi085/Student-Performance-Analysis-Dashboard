 Student Performance Analysis Dashboard

 Project Overview
This project analyzes student academic performance and provides meaningful insights through an interactive dashboard. It helps to understand subject-wise performance, student trends, and overall academic analysis.

 Technologies Used
- Python
- Pandas
- Excel
- Power BI
- Data Visualization

 Features
- Student performance analysis
- Subject-wise score comparison
- Identification of performance trends
- Interactive dashboard visualization
- Data-driven insights for better decision making

 Project Workflow
1. Data Collection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis
4. Dashboard Creation
5. Insight Generation

 Outcome
Created an interactive dashboard to visualize student performance patterns and help in understanding academic improvements.
